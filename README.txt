Display the Comexposium Catalogue on your website.

== Description ==

This plugin help you to display easily the Comexposium Catalogue and allow indexing bots to crawl and index exhibitors pages and product pages.

== Installation ==

1. Upload `catalogue-comexposium` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Where can I found the show id? =

You can ask your Comexposium contact to give you the id. Otherwise, you can send an email to dweb@comexposium.com.

= How many langages are available? =

For now, you can only use french or english version of the Catalogue.

= Can I have multiple Catalogue in my website? =

For now, you can only display one Catalogue per website. This is in order to simplify indexing bots can works andand do not complicate the search results about Catalogue contents.
